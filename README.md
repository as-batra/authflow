# AuthFlow – Full Stack Authentication System

AuthFlow is a full-stack web application that enables secure user authentication using JWT (JSON Web Tokens). It allows users to register, log in, and access protected routes with a clean and responsive UI.


---

## Tech Stack

* **Frontend:** React + Vite + Tailwind CSS
* **Backend:** Node.js + Express.js
* **Database:** MongoDB Atlas (Mongoose)
* **Authentication:** JWT + bcrypt

---

##  Features

* User Signup & Login
* JWT-based Authentication
* Protected Routes
* Password Hashing with bcrypt
* Full Stack Deployment (Vercel + Render)
* Responsive UI

---

## Deployment

* **Frontend:** Deploy on Vercel
* **Backend:** Deploy on Render
* Update environment variables accordingly
